Дош (Dosh) - Chechen Wordle

Дош is a localized version of the Wordle game created for the Chechen language.
The project is designed to encourage the use of the native language and learn some new words.

Features The game provides full native language support by utilizing a dictionary of Chechen words.
A specialized on-screen keyboard is included to handle specific Cyrillic characters.
It also tracks user progress  and offers translations for completed words to aid in vocabulary building.
